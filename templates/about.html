An Image Processing App

About This App

This is a simple web application for image processing built with Flask and OpenCV. The app allows you to upload an image and perform various operations such as converting the image to grayscale, detecting faces, and converting the image to different formats.

How to Use This App

To use this app, simply follow the steps below:

  1. Click on the "Choose File" button to select an image from your device.
  2. Select the desired operation from the drop-down menu.
  3. Click on the "Process Image" button to perform the selected operation on the uploaded image.
  4. The processed image will be displayed below along with the option to download the processed image in the desired format.

Technologies Used

This app was built using the following technologies:

  • Python
  • Flask
  • OpenCV
  • Bootstrap
